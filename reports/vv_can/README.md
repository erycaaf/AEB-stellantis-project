# reports/vv_can/ — V&V artefacts for the CAN Bus module

Cross-validation of `src/communication/aeb_can.{c,h}` against ISO 26262-6:2018
ASIL-D requirements.

- **Original author:** Renato
- **Cross-validator:** Lourenço
- **Report:** `Relatorio_Consolidado_VV_CAN.pdf` (built from the `.tex` source)

## How to reproduce everything

From the repository root:

```bash
make vv-can
```

This runs the four sub-targets (`mcdc-can`, `fault-can`, `memory-can`, `misra-can`)
and regenerates everything in this directory.

Toolchain required: `gcc-14`, `gcov-14`, `cppcheck 2.13+` with misra addon,
`valgrind 3.22+`, `lcov 2.0+`.

To rebuild the PDF report:

```bash
cd reports/vv_can
pdflatex Relatorio_Consolidado_VV_CAN.tex
pdflatex Relatorio_Consolidado_VV_CAN.tex   # second pass for cross-refs
```
